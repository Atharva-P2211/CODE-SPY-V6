## 🎮 What Is CodeSPY?

CodeSPY is **Among Us meets LeetCode**. Every round, all players solve the same coding challenge — but one submission is secretly generated by an AI (powered by Amazon Bedrock). Players study all the submissions and vote on which one they think was written by the machine.

**The twist?** The AI is getting smarter every round. It mimics human mistakes, skips type hints, adds casual comments, and adapts its style to the difficulty tier. Can you spot it?

## ✨ Features

### 🎯 Core Gameplay
- **4 Difficulty Tiers** — Rookie, Apprentice, Agent, Elite
- **Real-time multiplayer** — up to 20 players per room
- **22 unique challenges** — from FizzBuzz to LRU Cache
- **Smart AI submissions** — powered by Amazon Bedrock (Claude 3 Haiku)
- **Cinematic phase transitions** — Coding → Accusation → Voting → Debrief

### 🔥 Engagement Features
- **Streak multiplier** — 3 catches in a row = 3x points
- **Confidence betting** — bet 1x, 2x, or 3x before you vote
- **Accusation chat** — 30-second live debate before voting
- **Emoji reactions** — react to suspicious code in real time
- **Wildcard** — double or nothing for last-place players
- **MVP badges** — Best Deceiver 🎭, Sharpest Eye 🦅, Speed Demon ⚡
- **AI Glitch Reveal** — cinematic glitch animation when AI is unmasked
- **Share card** — copy your result and share it anywhere
- **XP + Level system** — persistent progression across sessions

### ☁️ AWS Integration
- **Amazon Bedrock** — Claude 3 Haiku generates a unique AI solution every round
- **Amazon DynamoDB** — global leaderboard + live game stats
- **AWS Lambda** — serverless backend for AI generation and leaderboard
- **Amazon API Gateway** — REST API connecting frontend to AWS services
- **Amazon Polly** — voice narration for phase transitions (Web Speech API fallback)

### 🎨 Design
- Dark terminal aesthetic — `#060912` background, neon accents
- Fonts: Bebas Neue, Rajdhani, JetBrains Mono
- Animated dot grid, scanlines, vignette overlay
- Syntax highlighting in voting view
- Spectator heatmap — shows which submissions are getting attention
- Live typing indicators during coding phase
- Fully responsive — mobile to desktop

## ⚙️ Setup

### Play Instantly (No AWS Needed)
The game works fully in local mode — just open `index.html` in any browser. Bots simulate other players, AI uses pre-written solutions.

### Enable AWS Features

**Prerequisites**
- AWS account (free tier covers everything)
- Python 3.8+
- `pip install boto3 awscli`

**Step 1 — Configure AWS credentials**
```bash
aws configure
# Enter your Access Key ID
# Enter your Secret Access Key
# Default region: ap-south-1
# Default output: json
```

**Step 2 — Run setup script**
```bash
python3 setup_aws.py
```
This automatically creates:
- DynamoDB tables (`codespy-leaderboard`, `codespy-stats`)
- IAM role with required permissions
- Lambda functions (AI generator + leaderboard)
- API Gateway with CORS configured
- Deploys everything to `ap-south-1` (Mumbai)

**Step 3 — Build with your API URL**
```bash
python3 build_v6.py --api-url https://YOUR_ID.execute-api.ap-south-1.amazonaws.com/prod
```

**Step 4 — Deploy**
```bash
# Rename output to index.html and push to GitHub
# Vercel auto-deploys on every push
```

## 🎮 How To Play

1. **Create a room** → choose your difficulty tier
2. **Share the room code** with friends (or play with bots)
3. **Code phase** — everyone solves the same challenge in the time limit
4. **Accusation phase** — 30-second live chat to debate submissions
5. **Vote** — pick which submission you think was written by AI (bet 1x–3x)
6. **Debrief** — AI is revealed with a glitch animation, scores calculated
7. **Play again** — streaks and multipliers carry over each round

## ☁️ AWS Services Used

| Service | Usage | Why |
|---------|-------|-----|
| **Amazon Bedrock** | Generate AI code submissions via Claude 3 Haiku | Fresh unique AI code every round — never the same solution twice |
| **AWS Lambda** | Serverless functions for AI + leaderboard | Pay per game, auto-scales to zero, no servers to manage |
| **Amazon DynamoDB** | Global leaderboard + game stats | Single-digit ms reads, scales to millions of players |
| **API Gateway** | REST API connecting frontend to Lambda | Managed HTTPS endpoints with CORS, no backend server needed |
| **Amazon Polly** | Voice narration for phase transitions | Dramatic text-to-speech with Web Speech API fallback |

## 🏆 Scoring

```
Base points × Streak multiplier × Bet multiplier × Wildcard bonus

Examples:
  Elite tier, 3 catch streak, 3x bet = 200 × 2x × 3x = 1,200 pts
  Agent tier, first catch, 1x bet  = 150 × 1x × 1x = 150 pts
  Rookie tier, 4 catch streak, 2x bet = 50 × 3x × 2x = 300 pts
```

## 🛡️ Security Notes

- Never commit AWS credentials to GitHub
- Use IAM roles with minimum required permissions in production
- API Gateway has CORS configured for your domain only in production
- DynamoDB records have 90-day TTL to keep costs minimal
